# Ying for Simple PHP Blog

本主题根据 [MagicBreeze/halo-theme-Ying](https://github.com/MagicBreeze/halo-theme-Ying) 移植，保留原主题的极简内容布局、个人信息区、文章列表、细线导航、深色模式和轻纹理背景。

适配内容：

- 首页文章列表与分页
- 文章、独立页面、分类、标签与归档
- 友情链接
- 原生评论、回复与表单状态
- 移动菜单、深色模式和返回顶部
- 当前博客的站点信息、作者头像、个人主页和自定义页面导航

Halo 专用的插件接口、PJAX、jQuery、瞬间、相册和设备模板没有移植。主题使用当前博客已有能力，不引入额外前端依赖。
